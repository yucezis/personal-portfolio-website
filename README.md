# ✨ Full-Stack Developer Portfolio – Zişan Yüce

Welcome to my personal portfolio project! This repository contains the source code for my portfolio website, designed to showcase my journey as a **Junior Full-Stack Developer** specializing in **.NET** and **Flutter**.

The project is built with a focus on **Clean Architecture principles**, **maintainability**, and a unique **Dark Academia aesthetic**.

---
💫 **Check out the live portfolio here:** [Click here to visit](https://zisan-yuce.vercel.app/)

---


## Key Features

- **Goal-Oriented Design**  
  Specifically tailored to showcase my technical projects, engineering background, and mandatory internship availability.

- **Advanced Theme Switcher**  
  Features a custom-designed Dark Mode inspired by *90s Grunge* and *Dark Academia* aesthetics (**Deep Burgundy & Carbon Black**) and a clean, professional Light Mode.

- **Multilingual Support (i18n)**  
  The architecture is prepared for multiple language options (**English** and **Turkish**) to cater to both local and global opportunities.

- **Fully Responsive**  
  Optimized for all screen sizes, from large desktop monitors to mobile devices, using modern **CSS Grid** and **Flexbox** layouts.

- **Technical Writing Showcase**  
  A dedicated **"Articles"** section to display my Medium publications and technical insights on .NET and Flutter.

- **Direct CV Access**  
  Integrated **"Download CV"** functionality for quick access by recruiters.

## Project Structure

Following the **Clean Architecture** philosophy, the project is organized into modular and maintainable files:
```
my-portfolio/
│
├── index.html # Semantic structure and content
├── style.css # Design tokens, themes (Dark/Light), and layouts
├── script.js # Interaction logic, animations, and theme persistence
├── assets/ # Images, icons, and PDF documents (CV)
└── README.md # Documentation
```
